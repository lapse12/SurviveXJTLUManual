# README

## 缘起

笔者听闻《上海交通大学的生存手册》这个项目由来以及，但是直到阅读之后，才深感相见恨晚。

这份详细讲解了生活、学习、工作的手册，始于2008年，由一群交大本科生编写，十几年来造福了无数交大和外校学子。然而，随着时间推移，一些内容可能已经过时，且和西浦的学校环境存在差异。因此，我希望尽自己的一份力，能让更多西浦学子看到这份手册；虽然我个人可能难以长期维护这个项目，但我希望通过这个开端，抛砖引玉，激发更多同学的共创。

西浦是时候融入一点开源精神了！

## 提issue

如果有同学对访谈集内内容持有怀疑意见，认为某篇访谈内容作者过于片面或者不属实请在issue区指出具体不合适的地方，我们随后会尽力寻找相似背景的同学进行二次review作出修改

## 如何PR

1. 在GitHub上fork本项目到个人账户
2. 在本地环境中进行修改
3. 提交变更并发起Pull Request
等待审核，如果通过，您的贡献将被合并到主分支
